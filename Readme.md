  - Add more sensors on the pinouts

### Libraries used

This demo uses the following libaries

- [Wire.h]
- [SSD1306Ascii]
- [SSD1306AsciiWire]
- [ESP8266WiFi]
- [Hash.h]
- [ESPAsyncTCP]
- [ESPAsyncWebServer]
- [DHT]

### Installation

Download the above libraries and place them in the libaries folder
Download this repo and run the .ino file
